# Efficiently architecting VQAs: code repository

![Repository overview](assets/Main_figure.png)

This repository contains the code accompanying the paper:

**Efficiently architecting VQAs: Expressibility--Trainability--Resources Pareto-Optimality**  
arXiv: https://arxiv.org/abs/2603.22142

## Overview

This repository provides the implementation used to study parametrized quantum circuit ansatzes under a multi-objective perspective, with emphasis on expressibility, trainability, and resource considerations.

The codebase currently contains:

- `ansatzes.py`: definitions of the variational ansatzes
- `vqe_analysis.ipynb`: notebook for computing and analyzing expressibility and trainability

## Repository purpose

The purpose of this repository is to provide a compact research artifact supporting the methodology and computational analysis presented in the paper.

## Requirements

The code relies on Python and PennyLane-based workflows. Additional dependencies may be specified in future updates.
